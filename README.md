# MMBOT (Team & Spectator Manager)

https://discord.gg/HMdfyDu3EC

MMBOT は、Fortnite や各種内戦を円滑に進めるためのチーム分け・観戦管理ボットです。
過去の履歴に基づいたバランスの取れたチーム分けと、絵文字（📺）による観戦者除外機能を備えています。

## 🚀 特徴
- **高精度なチーム分け**: 過去のチーム分け履歴を保存し、同じ人と連続して同じチームになりにくいロジックを採用しています。
- **📺 観戦者モード**: ボタン一つで名前に「📺」を付与。チーム分けの対象から自動で除外されます。
- **簡単操作**: スラッシュコマンド（`/setup`）で多機能パネルを設置し、誰でも直感的に操作できます。

- [Node.js](https://nodejs.org/) v16.11.0 以上がインストールされていること。
